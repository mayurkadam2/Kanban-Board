#  Kanban Board

A clean, dark-themed **Kanban Board** built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies, just the web fundamentals.

> Manage your tasks visually by moving them across **TODO → IN PROGRESS → DONE**.

---

##  Live Preview

👉 **[View Live Demo](https://mayurkadam2.github.io/Kanban-Board/)**

---

##  Features

-  **Add Tasks** — Create tasks with a title and description using the modal popup
-  **Three Columns** — Organize work into `TODO`, `IN PROGRESS`, and `DONE`
-  **Drag & Drop** — Move tasks between columns by dragging them
-  **Delete Tasks** — Remove tasks you no longer need
-  **Task Counter** — Each column shows a live count of tasks
-  **Progress Bar** — Visual bar that tracks your overall completion progress
-  **Local Storage** — Tasks are saved in the browser, so they persist on page refresh
-  **Dark Theme UI** — Sleek dark interface with color-coded columns

## 🚀 Getting Started

Even if you're completely new to coding, you can run this project in **3 steps**.

### Step 1 — Download the project

Click the green **`<> Code`** button on this page → **Download ZIP** → Extract the folder.

*OR* if you have Git installed, run:

```bash
git clone https://github.com/mayurkadam2/Kanban-Board.git
```

### Step 2 — Open the folder

Navigate into the extracted/cloned folder:

```
Kanban-Board/
├── index.html   ← Open this file
├── style.css
├── script.js
└── assets/
```

### Step 3 — Run it

Simply **double-click `index.html`** — it will open in your browser. That's it! No server needed. 🎉

---

##  How to Use

1. Click the **`+ Add New Task`** button in the top-right
2. Enter a **Task Title** and **Task Description**
3. Click **`Add Task`** — it will appear in the `TODO` column
4. **Drag the task card** to move it to `IN PROGRESS` or `DONE`
5. Hit the **trash icon** on a card to delete it
6. Watch the **progress bar** fill up as tasks move to Done!
